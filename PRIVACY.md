# Privacy Policy - YouTube Automation Tool

## Data Collection
This tool collects NO personal data. It only stores:

### Local Storage:
- X API credentials (encrypted in .env file)
- YouTube channel IDs for downloading
- Video files temporarily during processing
- Configuration settings in JSON files

### What We DO NOT Collect:
- Personal information
- Email addresses
- Browser history
- Location data
- Payment information

## API Usage
The tool only accesses:
- YouTube public videos (via yt-dlp)
- X API for posting content (no DMs, no following)

## Data Security
- All credentials stored locally only
- No data transmitted to third parties
- No analytics or tracking

## Third-Party Services
- YouTube (video download)
- X/Twitter (content posting)
- Google Drive (optional storage)
