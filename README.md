# YouTube Automation v2

## Copyright-Safe Video Automation Tool

### Features
- Downloads YouTube videos from educational channels
- Automatically creates copyright-safe edits
- Adds commentary, subtitles, watermarks
- Posts to X/Twitter automatically
- Google Drive integration

### Installation
1. Download `YouTubeAutomation.bat`
2. Run as administrator
3. Follow setup wizard
4. Configure Google Drive path

### Setup for X API
1. Apply for X Developer account
2. Create new app at https://developer.x.com
3. Use these URLs:
   - Website: https://github.com/[your-username]/YouTubeAutomation
   - Terms: https://github.com/[your-username]/YouTubeAutomation/blob/main/TERMS.md
   - Privacy: https://github.com/[your-username]/YouTubeAutomation/blob/main/PRIVACY.md
4. Add callback URL: `http://127.0.0.1:8080/callback`
5. Copy API keys to `.env` file
